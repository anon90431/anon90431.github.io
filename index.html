Identifying Knowledge Types and Developing a Reasoning Framework for Solving the Winograd Schema Challenge

Recently, there have been many works towards solving the Winograd Schema Challenge. These works mention the need of knowledge in solving the challenge but do not focus on identifying the different types of knowledge so that specific extraction techniques can be implemented to extract each such knowledge or/and type specific reasoning framework could be implemented to handle a particular type of knowledge. The major works are based on quasi-probabilistic correlation calculus, relevance theory and an ensemble of natural language processing techniques.


Though most of the works mention the requirement of knowledge in solving the problem, there has been very little work towards identification of the types of knowledge needed to correctly solve the challenge, so that different extraction intuitions/techniques can be implemented to extract each such knowledge or/and different reasoning framework could be used to handle a particular type of knowledge.


In this work, we identified 12 commonsense knowledge types such that about 97% of the Winograd Schema Challenge problems can be answered correctly by using at-least one instance of knowledge belonging one of the types. We also defined a logical reasoning framework to handle 10 out of 12 knowledge types, covering about 85% of the overall corpus. Later we provided an empirical evaluation of our reasoning framework by automatically extracting the required knowledge instances for 100 WSC problems and correctly solving them.

Following are the major contributions of this work.

  • Knowledge Types Identification
  • Logical Reasoning Framework
